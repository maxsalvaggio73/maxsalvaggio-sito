Assistente SQL Supabase

Configura il tuo Database:

Se non l'hai ancora fatto, accedi a Supabase, apri l'SQL Editor e incolla questo script per creare la tabella ed attivare l'approvazione automatica:

-- 1. Crea la tabella dei profili create table public.profiles ( id uuid references auth.users on delete cascade primary key, email text not null, is_approved boolean default false, is_admin boolean default false, -- Campo ruolo amministratore created_at timestamp with time zone default timezone('utc'::text, now()) not null ); -- 2. Abilita la lettura pubblica alter table public.profiles enable row level security; create policy "Lettura pubblica profili" on public.profiles for select using (true); create policy "Modifica profili per amministratori" on public.profiles for update using (true); -- 3. Crea la funzione di trigger create or replace function public.handle_new_user() returns trigger as $$ begin insert into public.profiles (id, email, is_approved) values (new.id, new.email, false); return new; end; $$ language plpgsql security definer; -- 4. Attiva il trigger create trigger on_auth_user_created after insert on auth.users for each row execute procedure public.handle_new_user();

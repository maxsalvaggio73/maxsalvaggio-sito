Assistente SQL Supabase

Configurazione Regole di Sicurezza Storage:

Dopo aver creato un bucket pubblico su Supabase denominato client-proofs, esegui questo script nell'SQL Editor per abilitare la gestione dei file agli amministratori e limitare la lettura ai rispettivi clienti:

-- ======================================================== -- REGOLAZIONI DI SICUREZZA PER LO STORAGE DEI PROVINI -- ======================================================== -- 1. Permette agli amministratori (is_admin = true) di gestire qualsiasi operazione create policy "Gestione totale file per amministratori" on storage.objects for all using ( bucket_id = 'client-proofs' and (select is_admin from public.profiles where id = auth.uid()) = true ); -- 2. Permette ai clienti approvati di leggere solo la propria cartella personale create policy "Lettura cartella personale per clienti" on storage.objects for select using ( bucket_id = 'client-proofs' and (select is_approved from public.profiles where id = auth.uid()) = true and name like (select email from public.profiles where id = auth.uid()) || '/%' );

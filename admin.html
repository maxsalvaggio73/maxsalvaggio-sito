Guida SQL e Regole di Sicurezza

Assicurati che la tabella profiles e le politiche di sicurezza dello Storage siano state configurate. Se necessario, esegui questa query in Supabase:

-- 1. Aggiunge le colonne necessarie alter table public.profiles add column if not exists is_admin boolean default false; alter table public.profiles add column if not exists file_order jsonb default '[]'::jsonb; -- 2. Permette agli amministratori di gestire lo storage client-proofs create policy "Admins can select files" on storage.objects for select using ( bucket_id = 'client-proofs' and (select is_admin from public.profiles where id = auth.uid()) = true ); create policy "Admins can insert files" on storage.objects for insert with check ( bucket_id = 'client-proofs' and (select is_admin from public.profiles where id = auth.uid()) = true ); create policy "Admins can delete files" on storage.objects for delete using ( bucket_id = 'client-proofs' and (select is_admin from public.profiles where id = auth.uid()) = true );

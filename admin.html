Assistente SQL Supabase

Configurazione Regole e Tabelle:

Esegui questo script nell'SQL Editor per aggiornare la tabella profiles con le colonne per l'amministrazione e l'ordine dei file, ed abilitare le politiche dello storage:

-- 1. Aggiunge le colonne is_admin e file_order (se non esistono) alter table public.profiles add column if not exists is_admin boolean default false; alter table public.profiles add column if not exists file_order jsonb default '[]'::jsonb; -- 2. Permette agli amministratori (is_admin = true) di gestire lo storage client-proofs create policy "Admins can select files" on storage.objects for select using ( bucket_id = 'client-proofs' and (select is_admin from public.profiles where id = auth.uid()) = true ); create policy "Admins can insert files" on storage.objects for insert with check ( bucket_id = 'client-proofs' and (select is_admin from public.profiles where id = auth.uid()) = true ); create policy "Admins can delete files" on storage.objects for delete using ( bucket_id = 'client-proofs' and (select is_admin from public.profiles where id = auth.uid()) = true ); -- 3. Permette ai clienti approvati di leggere solo la propria cartella create policy "Clients can select own files" on storage.objects for select using ( bucket_id = 'client-proofs' and (select is_approved from public.profiles where id = auth.uid()) = true and name like (select email from public.profiles where id = auth.uid()) || '/%' );
